# 🌦️ Weather-Forecast-ap

A simple and responsive Weather Forecast Web Application built using HTML, CSS, and JavaScript. This application allows users to fetch real-time weather data for any city using the OpenWeatherMap API.

## 🚀 Features

- Search weather by city name  
- Real-time temperature and weather details  
- Shows humidity and wind speed  
- Clean and modern UI design  
- Responsive layout for all devices  
- Fast and lightweight frontend project  

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  
- OpenWeatherMap API  

## ⚙️ How to Run Locally

1. Clone the repository  

```
git clone https://github.com/ayush765497/Weather-Forecast-app.git
```

2. Open the project folder  

```
cd Weather-Forecast-app
```

3. Open index.html in your browser
